# Metrics Reference

The operator exposes Prometheus metrics on the manager binary's metrics
endpoint (defaults to `:8443` HTTPS or `:8080` HTTP, see `--metrics-bind-address`
in [flags.md](flags.md)). All metric names are stable across patch releases;
new labels may be added.

Histogram bucket boundaries are tuned to the operator's 30s HTTP timeout —
anything in the 20-30s range indicates rate-limiter saturation or upstream
slowness, both of which warrant paging.
