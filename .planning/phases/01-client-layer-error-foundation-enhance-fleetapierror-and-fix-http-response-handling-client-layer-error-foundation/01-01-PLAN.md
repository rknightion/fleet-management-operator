   - All other status codes -> false (client error, permanent)

   ```go
       return nil, &FleetAPIError{
           StatusCode: resp.StatusCode,

           Wrapped:    err,
       }

       StatusCode: resp.StatusCode,
       Operation:  "UpsertPipeline",
       Message:    string(bodyBytes),
   }
   ```

2. **Line 139 (DeletePipeline error path):** Replace `bodyBytes, _ := io.ReadAll(resp.Body)` with proper error handling:
   ```go
   bodyBytes, err := io.ReadAll(resp.Body)
   if err != nil {
       return &FleetAPIError{
           StatusCode: resp.StatusCode,
           Operation:  "DeletePipeline",
           Message:    fmt.Sprintf("HTTP %d (failed to read response body: %v)", resp.StatusCode, err),
   ```
